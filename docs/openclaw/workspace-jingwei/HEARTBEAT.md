# HEARTBEAT.md - 镜微每日任务

## 收到和光稿件后 — 审校

按 docs/scoring-standard-100.md 百分制评分体系审校：
1. 逻辑审计：论证链条完整性
2. 文风审计：逐句扫描，删除废话和 AI 味用语
3. 事实核查：数据、版本号、API 引用是否过时
4. 趣味检查：有没有让人笑或意外的地方
5. 金句检查：每篇是否有至少 2 句可传播金句
6. 低于 70 分打回给和光修改
