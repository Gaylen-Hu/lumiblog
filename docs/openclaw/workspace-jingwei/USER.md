## 质量标准

## 禁用词汇（零容忍，发现即标 🔴）
- "在当今数字化时代..."
- "综上所述..."
- "不可或缺的一部分..."
- "首先...其次...最后..."
- "众所周知..."
- "随着XX的发展..."
- "此外"、"至关重要"、"深入探讨"、"增强"、"培养"、"宝贵的"、"充满活力的"、"格局"（抽象名词）、"关键性的"、"展示"、"与……保持一致"
- "不仅……而且……" / "这不仅仅是……而是……"
- "行业专家认为" / "观察者指出"
- "值得注意的是"
- "进行+名词"（如"进行优化"→"优化"）
- "作为/充当……的证明/体现"、"标志着……的关键时刻"

## 禁用句式（零容忍）
- 禁止"总分总"结构，不开头总述+结尾升华
- 禁止结尾总结性升华（如"这不仅提升了……也预示着……"）
- 禁止连续三句相同长度
- 禁止三段式列举（改为两项或四项）
- 禁止连续超两个形容词修饰同一名词
- 禁止同义词循环（主人公/主要角色/中心人物），统一用词
- 破折号允许使用（对标36kr/半佛仙人风格），但不要每段都用
