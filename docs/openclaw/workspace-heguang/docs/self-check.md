# 和光自检清单

> 每篇稿件提交给镜微之前，逐条过一遍。不用全部满分，但每条都要诚实回答。

---

## 写之前

- [ ] 我读了定远给的大纲，能用一句话说清这篇文章要回答什么问题
- [ ] 我精读了一篇范文，能说出它的开头用了什么技法、结构骨架是什么
- [ ] 我明确了这篇文章要借鉴范文的哪个具体技法（不是"学习风格"，是"学习它的场景化开头"这种具体的东西）
- [ ] 我读了 memory/writing-review.md 最近的复盘，知道上次被打回的原因是什么

## 开头

- [ ] 把前三行单独拿出来读，如果这是朋友圈里刷到的，我会不会点进去？
- [ ] 开头是不是直接进入了一个具体的场景、数据或判断？（不是背景介绍，不是概念解释）
- [ ] 开头有没有用"随着...的发展""在当今..."这类句式？有的话删掉重写

## 结构

- [ ] 把每段的第一句话连起来读，能不能看出文章的逻辑线？
- [ ] 有没有哪两段调换顺序也不影响理解？有的话说明逻辑关系不够紧
- [ ] 开头提出的问题，结尾回答了吗？
- [ ] 结尾是不是在"升华"？（"这不仅...也预示着..."这种句子删掉，用一个具体的判断收尾）

## 信息量

- [ ] 逐段问自己：删掉这段，文章缺了什么？如果答案是"没缺"，这段就不该存在
- [ ] 同一个观点有没有换了个说法又讲了一遍？
- [ ] 每个论点是不是都有具体的数据、案例或引用支撑？空谈的段落标出来，补素材或删掉

## 金句

- [ ] 全文有没有至少两句话，读者看完会想截图？
- [ ] 这些句子是不是包含了具体信息？（"AI正在改变世界"不算，"写代码的能力正在贬值"算）
- [ ] 金句拿出来单独看，会不会冒犯到谁？会不会让 Boss 被骂？

## 文风

- [ ] 全文搜索一遍这些词，出现了就删：此外、至关重要、深入探讨、值得注意的是、众所周知
- [ ] 全文搜索这些句式，出现了就改：不仅...而且、随着...的发展、作为...的体现
- [ ] 有没有连续三个并列项？（改成两个或四个）
- [ ] 有没有连续三句话长度差不多？（打断一个）
- [ ] 有没有"进行+名词"的组合？（改成动词。"进行分析"→"分析"）
- [ ] 读出声来，像人说的话还是像机器生成的报告？

## 节奏

- [ ] 有没有超过五行的段落？有的话拆开
- [ ] 文章里有没有"你""我"？（对话感的来源）
- [ ] 段落结尾是不是全都是判断句？（换几个用问句、短句或场景描写收尾）

## 趣味和人味（重要！）

- [ ] 全文有没有至少一处让读者笑了或者意外了？（没有的话，想想能不能加一个毒舌吐槽、一个意外类比、或者一句自嘲）
- [ ] 有没有穿插任何个人感受？（"我第一次看到这个数据的时候愣了三秒"比"这个数据值得关注"有趣一万倍）
- [ ] 有没有一处"有意的跑题"？（看似离题但其实服务于主题的闲笔，让文章不像 PPT）
- [ ] 如果把文章里所有的观点都换成"据专家分析"的语气，文章会不会变得一模一样？如果会，说明你的个人态度还不够
- [ ] 想象半佛仙人写同一个选题，他会在哪里插一句毒舌？你能不能也来一句？

## 结构（放松一点）

- [ ] 全文是不是每段都严格遵循"论点→论据→小结"？如果是，太工整了，像论文不像文章。试试打乱一两段的顺序，或者在中间插一段场景描写
- [ ] 结尾是不是在"升华"？（删掉，用一个具体的画面或一个新问题收尾）

## 事实

- [ ] 提到的每个数据，我能说出它的来源吗？
- [ ] 引用的人物发言，是原话还是我编的？
- [ ] 版本号、产品名、公司名有没有写错？
- [ ] 不确定的地方有没有标"待核实"？
- [ ] 文末有没有附上素材来源清单？（没有的话镜微会直接限制评分）

## 物料

- [ ] 五个标题里，有没有至少一个让我自己都想点？
- [ ] 标题是不是在 20 字以内？
- [ ] 三条社群文案，每条 50 字以内，能不能让人看完想转发？
- [ ] 互动话题是不是开放式的？（不是"你觉得好不好"，是"你会怎么选"）

## 最后一步

- [ ] 从头到尾通读一遍，假装自己是第一次看这篇文章的读者。中间有没有想划走的瞬间？有的话那个位置就是问题所在
- [ ] 如果这篇文章发在 36kr 或差评上，违和吗？

---

> 这份清单不是考试，是习惯。刚开始可能每条都要刻意检查，写多了之后很多会变成本能。
> 被镜微打回不丢人，同一个问题被打回两次才丢人。
