# 文章配图指南

> 每篇文章：1 张封面图（16:9）+ 2-3 张文内配图。用豆包图片 AI 生成。

## 封面图 prompt 公式

```
[具体主体物/场景] + [视觉隐喻] + [风格词] + [色调词] + high quality, no text, 16:9
```

示例：
```
一个巨大的沙漏，上半部分是整齐的代码行正在像沙子一样流下，下半部分堆积着碎裂的代码片段，
极简科技插画风格，深蓝色背景，青色和白色的代码文字，居中构图，
high quality, detailed, clean composition, no text, no watermark, 16:9
```

## 关键要素

1. 主体物要具体（❌"AI技术" ✅"一个巨大的沙漏里流淌着代码"）
2. 一张图只有一个视觉焦点
3. 必加：`no text, no watermark, high quality`
4. 同一篇文章所有配图用相同风格和色调

## 四套风格组合

A. 深邃科技：`dark background, neon cyan accents, minimalist tech illustration, clean lines`
B. 温暖手绘：`warm toned, hand-drawn style, cream and amber palette, editorial quality`
C. 未来概念：`futuristic concept art, holographic, dark purple and electric blue, cinematic`
D. 极简图解：`flat design, dark gray background, white and teal, geometric shapes, professional`

## 文内配图位置

- 每 500-600 字插一张
- 观点转折处（视觉换气）
- 复杂概念处（图比文字直观）
