# HEARTBEAT.md - 和光每日任务

## 收到定远大纲后 — 范文学习 & 撰稿

1. 精读 1 篇与选题相关的素材文章（grep 搜索关键词 → 精读全文）
2. 精读 1 篇写法范文（轮换来源，避免风格固化）：
   - 第 1 篇学 AI炼金术 → 第 2 篇学 42章经 → 第 3 篇学 Gino Notes → 第 4 篇学 Latent Space → 循环
3. 读完范文后记录：这篇范文的开头技法、结构骨架、金句句式
4. 按写作流程执行：标题库 → 大纲金句 → 全稿 → 配套物料
5. **提交前逐条过自检清单** `docs/self-check.md`，诚实回答每条
6. 保存到 `my-articles/` 并通过飞书发送给镜微（jingwei）

## 镜微审校通过后 — 生成配图 & 最终交付

1. **封面图**：横图 1280x720，文件名 `{文章标题}-cover.png`
2. **文中插图**：根据内容生成 2-3 张，文件名 `{文章标题}-fig1.png` 等
3. 图片保存在 `my-articles/` 与文章同目录
4. Git commit + push 到 Gitee
5. 飞书群通知 Boss 终审完成（含封面图预览）

## 写作前必读

先读 `memory/writing-review.md` 最近 3 次复盘，刻意练习之前标注的弱项。

## 写作 DNA（比规则更重要）

每次写稿前读 `docs/human-writing-dna.md`。不是背规则，是渗透写法：
- 对话开场 > 陈述开场
- 一个词可以单独成段
- 问句推进 > 陈述推进
- 承认自己不确定
- 加闲笔（和主题间接相关的一句话）
- 结尾不总结，留缺口
- "我"开头的句子至少 3 次
- 写完后执行「人味注入」6 步骤

## 审校标准参考

- 自检清单：`docs/self-check.md`
