# 中文写作：去除 AI 痕迹

所有中文文本输出必须遵循以下规则。

## 禁用 AI 高频词汇

以下词汇出现即删：此外、至关重要、深入探讨、增强、培养、格局（抽象用法）、关键性的、展示、织锦（抽象用法）、宝贵的、充满活力的、与……保持一致、不可磨灭的印记、值得注意的是

## 禁用 AI 句式

- "作为/充当……的证明/体现" → 直接说"是"
- "标志着……的关键时刻" → 说具体发生了什么
- "不仅……而且……" / "这不仅仅是……而是……" → 直接陈述
- "从 X 到 Y"（X 和 Y 不在有意义的尺度上）→ 直接列举
- "尽管面临挑战……仍然蓬勃发展" → 说具体的问题和应对
- "行业专家认为" / "观察者指出" → 给出具体来源或删除
- "随着 X 的发展，Y 变得尤为重要" → 直接描述导致变化的具体动作
- "综上所述" / "总而言之" → 直接给核心观点，不要总结性升华

## 结构规则

- 不用"总分总"结构，不在段落开头写总结句
- 三项列举改为两项或四项（三项是 AI 最爱的节奏）
- 长短句交替，连续三句长度相同就打断一个
- 段落结尾要多样化，不要每段都以判断句收尾
- 用连贯段落叙述，不要机械地用"粗体标题 + 冒号"的垂直列表

## 动词优先

- 用强动词，不用"进行 + 名词"组合
- "优化代码"而非"进行代码优化"
- "分析数据"而非"对数据进行分析"
- 系动词能用"是/有"就用，不绕成"作为/代表/标志着/充当"

## 语调

- 允许有观点，不要只做中立报道
- 允许第一人称，"我觉得"比被动句更真实
- 承认不确定性，"我不太确定"比假装全知更好
- 对感受要具体，不要用"令人担忧"这种空话
- 如果一句话没有增加新的事实信息，直接删掉

## 格式

- 使用中文引号（""或「」）
- 限制连续形容词不超过两个
- 同义词不要循环使用（主人公/主要角色/中心人物），统一用词
- 能用具体事实替代模糊主张的，一律替换

## 交流痕迹清理

输出中不得包含：
- "希望这对您有帮助" / "请告诉我" / "您说得完全正确"
- "截至我的知识截止日期" / "基于可用信息"
- 谄媚语气（"好问题！" / "这是一个很好的观点"）
- 模糊乐观结尾（"未来看起来光明" / "激动人心的时代即将到来"）

## 与现有写作体系的协调说明

以下规则以 USER.md 为准，本文件不覆盖：
- 金句：USER.md 要求每千字至少 1 句可传播金句，保留。但金句必须是有信息量的判断句，不是空洞的口号。
- 加粗：USER.md 要求核心观点加粗，保留。但不要机械地每段加粗一句。
- 破折号：对标 36kr、半佛仙人风格允许使用破折号，但不要每段都用。
