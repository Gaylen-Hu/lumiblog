# 博客系统设计稿

基于 byte&beyond 项目风格的博客系统 UI 设计参考。

## 设计风格

- **极简主义**: 大量留白，清晰的视觉层次
- **色彩方案**: 主色 #111111（深黑），辅助色 #555555（灰色），强调色 蓝色渐变
- **圆角设计**: 大圆角卡片（24px-40px），按钮使用 full rounded
- **动效**: 微妙的悬停效果，平滑过渡动画
- **字体**: 大标题粗体，正文轻量级

## 页面结构

1. **公开博客 (blog-web)** - 面向访客
2. **管理后台 (admin-web)** - 内容管理

## 文件说明

- `pages/` - 各页面设计组件
- `components/` - 可复用 UI 组件
- `tokens.ts` - 设计令牌（颜色、间距、字体）
