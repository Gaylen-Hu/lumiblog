# 需要添加的接口

## 博客前端接口

### 获取文章列表

```
GET /api/articles
```

**说明：** 获取公开发布的文章列表，支持分页和筛选

**查询参数：**
- `page` (number, 可选) - 页码，默认 1
- `pageSize` (number, 可选) - 每页数量，默认 10
- `category` (string, 可选) - 分类筛选
- `tag` (string, 可选) - 标签筛选

**响应：**
```json
{
  "data": [
    {
      "id": "string",
      "slug": "string",
      "title": "string",
      "excerpt": "string",
      "author": "string",
      "date": "string",
      "readTime": "string",
      "category": "string",
      "imageUrl": "string",
      "tags": ["string"]
    }
  ],
  "total": 100,
  "page": 1,
  "pageSize": 10,
  "totalPages": 10
}
```

---

### 获取文章详情

```
GET /api/articles/:slug
```

**说明：** 根据 slug 获取单篇文章的完整内容

**响应：**
```json
{
  "id": "string",
  "slug": "string",
  "title": "string",
  "excerpt": "string",
  "content": "string",
  "author": "string",
  "date": "string",
  "readTime": "string",
  "category": "string",
  "imageUrl": "string",
  "tags": ["string"]
}
```

**错误响应：**
- `404 Not Found` - 文章不存在

## 模板

### 接口名称 🔒

```
METHOD /path
```

**说明：** 接口描述

**请求体：**
```json
{
  "field": "value"
}
```

**响应：**
```json
{
  "field": "value"
}
```

**错误响应：**
- `400 Bad Request` - 参数错误
- `401 Unauthorized` - 未认证
